# Base Styles

Base SCSS utilities and variables for FinPress.

Note: This package requires [Dart Sass](https://www.npmjs.com/package/sass) to compile. When using this package, make sure you are using Sass version 1.23.0 or later, which is based on Dart Sass.

## Installation

Install the module

```bash
npm install @finpress/base-styles --save-dev
```

## Usage

In your application's SCSS file, include styles like so:

```scss
@use '@finpress/base-styles/colors';
@use '@finpress/base-styles/variables';
@use '@finpress/base-styles/mixins';
@use '@finpress/base-styles/breakpoints';
@use '@finpress/base-styles/animations';
@use '@finpress/base-styles/z-index';
@use '@finpress/base-styles/default-custom-properties';
```

Make sure to use namespaces when accessing utilities, variables, functions, etc. For example:

```scss
.selector {
	color: colors.$gray-300;

	@include mixins.break-medium() {
		font-size: variables.$font-size-large;
	}
}
```

### CSS Admin Themes

The package also includes a built version of the output generated by the `finpress-admin-schemes()` mixin which can be used directly in your project. The CSS file is located at `node_modules/@finpress/base-styles/build-style/admin-schemes.css`.
