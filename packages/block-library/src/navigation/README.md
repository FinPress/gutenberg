# Navigation block

The Navigation block is a container block for the collection of blocks that allow visitors to get around your site. See also the Navigation Link block and Navigation Submenu block.

## CSS Classes

The structural CSS for the navigation block targets generic classnames across menu items of multiple types including those automatically generated by the Page List block. Here are some of the notable classnames and what they are used for:

-   `.fp-block-navigation__submenu-container` is applied to submenus to main menu items.
-   `.fp-block-navigation-item` is applied to every menu item.
-   `.fp-block-navigation-item__content` is applied to the link inside a menu item.
-   `.fp-block-navigation-item__label` is applied to the innermost container around the menu item text label.
-   `.fp-block-navigation-item__description` is applied to the innermost container around the menu item description.
-   `.fp-block-navigation__submenu-icon` is applied to the submenu indicator (chevron).
