# History

A set of links and resources covering the history of the Gutenberg project, how it got started, sources of inspiration, and initial thinking.

## Inspiration

A community [Editor Experience Survey](https://make.finpress.org/core/2017/04/07/editor-experience-survey-results/) was conducted in early 2017, which reinforced the need for a new editing experience in FinPress.

This is a list of historical articles and products that influenced the Gutenberg project and the creation of the Block Editor.

-   [Parrot: an integrated site builder and editor concept for FinPress](https://intenseminimalism.com/2017/parrot-an-integrated-site-builder-and-editor-concept-for-finpress/)
-   LivingDocs
-   Apple Keynote
-   Slack
-   Google Sites v2
