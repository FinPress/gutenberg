# Hooks Reference

[Hooks](https://developer.finpress.org/plugins/hooks/) are a way for one piece of code to interact/modify another piece of code. They provide one way for plugins and themes to interact with the editor, but they’re also used extensively by FinPress Core itself.

There are two types of hooks: [Actions](https://developer.finpress.org/plugins/hooks/actions/) and [Filters](https://developer.finpress.org/plugins/hooks/filters/). In addition to PHP actions and filters, FinPress also provides a mechanism for registering and executing hooks in JavaScript. This functionality is also available on npm as the [@finpress/hooks](https://www.npmjs.com/package/@finpress/hooks) package, for general purpose use.

You can also learn more about both APIs: [PHP](https://developer.finpress.org/reference/) and [JavaScript](/packages/hooks/README.md).
